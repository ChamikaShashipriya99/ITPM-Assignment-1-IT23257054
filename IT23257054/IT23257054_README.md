# Singlish to Sinhala Transliteration - Playwright Automation Tests
# IT3040 - ITPM Assignment 1 (Option 1)

## 📋 Project Description

This project contains automated test cases for the Singlish to Sinhala transliteration system available at [PixelsSuite Chat Translator](https://www.pixelssuite.com/chat-translator). The test suite focuses on identifying **50 distinct failure scenarios** where the system incorrectly converts chat-style Singlish into Sinhala.

The automation is built using **Playwright (Python)** as part of the IT3040 ITPM Assignment 1 requirements.

## 🎯 Test Scope

The test suite covers **24 categories** of Singlish input as specified in the assignment guidelines, with at least 2 test cases per category. All 50 test cases are designed to highlight system weaknesses (Negative Tests).

### Coverage Categories:
- Question and Command forms
- Greetings, Requests, and Responses
- Repeated and Compound words
- Phonetic variations (Vowel length, Nasalization, Retroflex consonants)
- Mixed language content (English words, Abbreviations, Clipped forms)
- Specialized formats (Place names, Person names, Numbers, Currency, Time, Dates, Units)
- Slang, Casual phrasing, and Online identifiers (URLs, Emails)
- Emojis and very long inputs (≥300 characters)

## 🔧 Prerequisites

- **Python 3.11+**
- **pip**
- **Google Chrome** (or Playwright Chromium)

## 📦 Installation

1. Install required Python packages:
   ```bash
   pip install playwright openpyxl
   ```

2. Install Playwright browser binaries:
   ```bash
   python -m playwright install chromium
   ```

## 🚀 Running Tests

To execute the automation script and update the Excel results:

```bash
python IT23257054_test_automation.py --excel "IT23257054_Test_cases.xlsx" --url "https://www.pixelssuite.com/chat-translator" --wait-ms 5000 --save-every 1
```

### Script Arguments:
- `--excel`: Path to the test cases Excel file.
- `--url`: Target application URL.
- `--wait-ms`: Delay after typing to allow for real-time transliteration (default 5000ms).
- `--save-every`: Frequency of saving results to Excel (default 1).

## 📊 Understanding Results

The Excel file `Assignment 1 - Test cases.xlsx` contains:
- **Expected Output**: The correct Sinhala transliteration.
- **Actual Output**: The result produced by the live system.
- **Status**: `FAIL` indicates that the system produced an incorrect result, satisfying the assignment requirement to find 50 failing cases.
- **Singlish input types covered**: The specific category from Appendix 1.
- **Rationale**: Explanation of why the input was expected to fail or what it tests.
